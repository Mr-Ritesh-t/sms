# **App Name**: CampusFlow

## Core Features:

- Student Profiles: Create and manage student profiles with enrollment details, contact information, and academic history.
- Teacher Profiles: Create and manage teacher profiles with assigned courses and contact information.
- Course Management: Create, update, and manage course details, including schedules, descriptions, and assigned teachers.
- Grade Submission: Allow teachers to submit grades for students in their assigned courses.
- Grade Viewing: Enable students to view their grades and academic performance.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to convey trust and stability.
- Background color: Light gray (#EEEEEE) for a clean and neutral backdrop.
- Accent color: Slightly darker blue (#42A5F5) for interactive elements and highlights.
- Body and headline font: 'Inter' for a modern, neutral, and readable interface.
- Use clear and consistent icons for navigation and actions.
- Clean and organized layout with a focus on usability and easy access to information.
- Subtle transitions and feedback animations for a smooth user experience.
